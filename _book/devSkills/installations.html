Installations

Sublime Text or Atom - Basic text editors that can grow with you.
Chrome - Not just a browser, also an indispensible piece of your development environment with it's ChromDev console.
Atomic Chrome - This will take a little more time to get used to, but could be worth the effort.
Node.js - For running js files in terminal only, don't worry about node servers yet.

Sublime Linting:

  • Sublime Package Control
  • Sublime Linter
  • jsHint
  • ESLint

visual code seems cool, give it a try too.
